## Image Guidelines

- All images should be optimized for web (compress to <200KB)
- Use descriptive filenames with keywords
- All courthouse images are geotagged with EXIF GPS data
- Recommended format: JPG for photos, PNG for graphics
- Courthouse images: 800x400px recommended
- Attorney headshot: 400x400px minimum
